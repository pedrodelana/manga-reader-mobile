This project is only for study purposes.

## Quick summary

A mobile app built with to list manga by consuming the **MangaDex** public API.

## Quick start

```bash
npm i
npm expo start
```

After that, in the Expo terminal, choose:

- `a` to open on Android
- `i` to open on iOS
- `w` to open in the browser
- `j` to open debbuger
